Core Product Description

1) User has list of Vendors to reach out:
      Vendor Name | emails | Report Name
3) User sends link to webpage to fill out form (responses compiles under Report Name)
4) Webpage has form that link recipients fill out; response, once submitted, is converted in a specific format and emailed to inbox
5) Script runs through inbox for specific Report Name filter and compiles it into CSV
6) User opens Csv in excel to obtain report
